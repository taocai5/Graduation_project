## 第三部分：PSO-KMeans 算法创新（3-4 页）

### 第 1 页：算法设计思路
**文字说明**：
- 传统 K-Means 的局限：对初始中心敏感
- PSO 的优势：全局搜索能力强
- 融合策略：PSO 寻优 + K-Means 精细化

**伪代码或流程图**：
```
1. 初始化粒子群（每个粒子代表 k 个聚类中心）
2. PSO 迭代优化（30 轮）
   - 更新速度和位置
   - 计算适应度（SSE）
   - 更新个体最优和全局最优
3. 使用全局最优作为 K-Means 初始中心
4. K-Means 局部精细化
```

### 第 2 页：算法性能对比 - SSE
**图表**：`2_算法对比_SSE.png`

**要点**：
- 三种算法在不同 k 值下的 SSE 对比

- 验证了算法的有效性

### 可能的提问
1. **为什么选择 PSO-KMeans？**
   - 答：传统 K-Means 对初始中心敏感，PSO 可以全局搜索最优初始点

2. **数据增强是否合理？**
   - 答：通过添加高斯噪声，保持数据分布特性，扩充样本量

3. **如何验证聚类效果？**
   - 答：使用 SSE、轮廓系数、Davies-Bouldin 指数等多个指标

4. **学业诊断的依据是什么？**
   - 答：基于聚类中心的特征值，结合教育学经验制定规则

5. **下一步工作重点？**
   - 答：论文撰写、英文翻译、实验补充
